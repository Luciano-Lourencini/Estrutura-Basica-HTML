Estrutura do HTML

A estrutura básica do HTML é composta de 5 tags, sendo elas:

  • <!DOCTYPE html>

    Serve para especificar que o código é HTML5

  • <html>

    Indica que qualquer coisa dentro dele é um código HTML

  • <head>

    Possui dados da página que não serão exibidos na janela principal

  • <title>

    Título da página

  • <body>

    Corpo da página. Tudo que está aqui será exibido na página principal

Para de quebrar linhas, deve-se usar a tag <br>

Para   espaçar   as palavras, deve-se utilizar   &nbsp;

Outras Marcações Interessantes

Algumas marcações interessantes são:

       

Listas

Todas as listas recebem os seus itens pela tag <li>, as diferenças são feitas na sua tag de abertura


Ordenadas

Itens da lista são enumerados
Para se criá-la, deve se usar a tag de abertura <ol>
Exemplo:

  1. Um
  2. Dois
  3. Três


Não Ordenadas

Itens da lista não são enumerados
Para se criá-la, deve se usar a tag de abertura <ul>
Exemplo:

  • Pão
  • Requeijão
  • Manteiga


Listas Aninhadas

Uma lista dentro de outra
Exemplo:

  • Salgados
    • Coxinha
    • Esfirrinha
  • Doces
  • Brigadeiro
  • Beijinho
